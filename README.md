# Movie-trailer
# 🎬 Movie Trailer Generator

An AI-powered Movie Trailer Generator built using Python and Google Colab. This project generates cinematic movie trailers by combining AI-generated scenes, voice-over narration, background music, subtitles, and video editing techniques into a final trailer.

## 🚀 Features

* 🎥 Automatic trailer generation
* 🤖 AI-generated narration
* 🎙️ Text-to-Speech voice-over
* 🎵 Background music integration
* 📝 Automatic subtitle generation
* 🎬 Scene transitions and video editing
* ☁️ Runs on Google Colab

## 🛠️ Technologies Used

* Python
* Google Colab
* FFmpeg
* MoviePy
* OpenCV
* gTTS / Edge-TTS
* NumPy
* Pillow

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/JATINGOYAL18/Movie-trailer.git
```

2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Open the notebook in Google Colab or Jupyter Notebook.

4. Run all cells in order.
